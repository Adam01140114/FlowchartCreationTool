Configure your OpenAI API key to enable AI-powered legal guidance

How to get your API key:

  1. Go to OpenAI Platform
  2. Sign up or log in to your account
  3. Navigate to the API section
  4. Click "Create new secret key"
  5. Copy the key (starts with 'sk-')
Security Note: Your API key is stored locally in your browser and never shared with our servers. Only OpenAI receives your key for API calls.
